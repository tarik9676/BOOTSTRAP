Welcome to the website. Here you can learn web development. You can build a website as you wish even like an application. So there is a lot of fun in learning web development.


You need to learn HTML5, CSS3, Bootstrap, JavaScript, jQuery, PHP, MySQL and WordPress. Here HTML5 and CSS3 help to build the structure of webpage. Bootstrap work as the frame of CSS. JavaScript and PHP are the programing language. jQuery work as the frame of JavaScript. MySQL is used for the database of the website. Wordpress is the easiest way to build a webpage. But all of this tools are mainly categorized in two section. Those are front-end design and back-end design.

  • Front-end design tools

    1. HTML5
    2. CSS3
    3. Bootstrap
    4. JavaScript
    5. jQuery

  • Back-end design tools

    1. PHP
    2. MySQL
    3. WordPress

We need a editor to write those languages except MySQL and Wordpress. Anyone can write a language in Notepad. You can also make a webpage with MS Word. But it is not even good idea to use Notepad and MS Word. Till 2018 most popular editors are Visual Studio Code, Atom, Sublime Text 3, Notepad++ etc. I will advise you to keep at least two editors and those are Atom and Visual Studio Code or as you feel comfortable.
